# Credit Card Fraud Analysis

## Project Overview
A complete **Credit Card Fraud Analysis** pipeline that extracts, processes, and visualizes transactional data to identify fraudulent activities. Designed to deliver actionable insights for fraud detection.

## Key Skills & Tools
- **SQL:** Data extraction and aggregation  
- **Python:** Automation and CSV exports (`pandas`, `mysql-connector-python`)  
- **Data Analysis & Visualization:** Power BI dashboards, KPI tracking  
- **ETL & Reporting:** End-to-end workflow from raw data to insights  

##  Features
- Tracks **total transactions**, **fraud transactions**, and **fraud rate**  
- Visualizes patterns by **merchant category, transaction hour, device trust, and location mismatch**  
- Automates data extraction and preprocessing for dashboard-ready CSVs  

## Future Enhancements
- Add **real-time fraud detection** using machine learning  
- Enhance dashboard interactivity and dynamic KPIs  
- Extend analytics with **predictive modeling**
